Fitting differential equations to data
- A simple introduction

Scenario: An emerging epidemic

Knowing future development is crucial to assess hospital bed requirements.

An SIR model differential equation can make predictions

Here, the model can be simulated from known initial conditions.

Problem: epidemic-specific parameters are unknown

The infection rate (β) and recovery rate (γ) affect predictions.

Solution: Fit parameters to known data

Roughly: Simulate model for many proposed parameter sets, select the best fitting one.

(In practice, good strategies for finding the optimal set exist)

We can now make predictions

If our fit was correct, we should be able to predict the epidemic's trajectory
